## 📌 Sobre a Disciplina

Nesta disciplina, vamos trabalhar com **desenvolvimento web utilizando PHP**.

### 🐘 O que é PHP?

O **PHP (Hypertext Preprocessor)** é uma linguagem de programação voltada principalmente para o **desenvolvimento de aplicações web no lado do servidor**.

Isso significa que o PHP é executado no servidor e é responsável por:

- Processar dados enviados por formulários  
- Conectar e manipular bancos de dados  
- Controlar regras de negócio  
- Gerar páginas web dinamicamente  

Grande parte dos sites e sistemas web do mundo utilizam PHP em sua estrutura.

---

## 🌐 APIs e Integrações

Durante a disciplina, também trabalharemos com **APIs (Application Programming Interfaces)**.

As APIs permitem que diferentes sistemas se comuniquem entre si. Por exemplo:

- Consultar dados de um serviço externo  
- Enviar informações para outro sistema  
- Integrar aplicações distintas  

Você aprenderá como consumir e criar APIs, entendendo como ocorre a troca de dados entre aplicações web.

---

## 🐳 Ambiente de Desenvolvimento com Docker

Para facilitar o desenvolvimento e garantir que todos trabalhem em um ambiente padronizado, utilizaremos o **Docker**.

O Docker permite criar ambientes isolados (containers) que já possuem:

- Servidor web  
- PHP configurado  
- Banco de dados  
- Demais dependências necessárias  

Isso evita problemas como:
- “Na minha máquina funciona”
- Conflitos de versões
- Dificuldades de configuração

Com o Docker, o ambiente de desenvolvimento se torna mais organizado, portátil e profissional.
